# 3DARMY

## ゲームの内容
戦車を操り、広大な砂漠で敵を殲滅するアクションシューティング(WIP)

敵を倒すと上部のゲージが減少し、減らし切るとゲームクリア(WIP)

画面左下のHPゲージは敵の攻撃に当たると減少、0になるとゲームオーバー(WIP)

![画面イメージ](docs/images/game_image01.png)

## 操作説明
上下矢印キーで前進・後退

左右矢印キーで左右旋回

マウス操作で照準、左クリックで射撃(WIP)
